An Implementation of Repository Pattern with ASP.NET Core 3.1 Web MVC. With this Open-Source Template, we create an abstraction layer between the data access and the business logic layer of an application. By using it, we are promoting a more loosely coupled approach to access our data from the database. Also, the code is cleaner and easier to maintain and reuse. Data access logic is in a separate class, or sets of classes called a repository, with the responsibility of persisting the applicationâ€™s business model.

Check out the [GitHub](https://github.com/raihanM95/RepositoryPattern) to learn more about this Open Source Project.

### Download the Extension
Make sure Visual Studio 2019 is installed on your machine with the latest SDK.
[Download from Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=RaihanMahmud.RepositoryPattern). Install it on your machine.
